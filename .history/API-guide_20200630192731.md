### URIs_list

#### Single example:

```
{
    "patients": "https://blood-dontaion.herokuapp.com/api/v1/patients/",
    "donations": "https://blood-dontaion.herokuapp.com/api/v1/donations/",
    "users": "https://blood-dontaion.herokuapp.com/api/v1/users/",
```

| ---------- | ---------- | ------- |
